# sales-funnel

Default funnel for inbound leads.

1. **Qualify**: ask budget, timeline, decision-maker.
2. **Demo**: schedule a 20-min demo if budget ≥ $1k/mo.
3. **Proposal**: send within 24h of the demo.
4. **Follow-up**: 3 touches over 10 days, then archive.

If the lead came from a `sofia` escalation, read the linked session file before reaching out — don't make the customer repeat themselves.
