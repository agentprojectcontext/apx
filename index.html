What APX provides

⚡

Daemon + CLI

Local HTTP daemon auto-starts. apx run spawns a full agent session in any runtime.

🔀

Multi-runtime

Claude Code, Codex, OpenCode, Aider. Route to any runtime, fall back automatically.

🗂️

Filesystem-first

No database. All agent state — memory, sessions, messages — lives in .apc/.

📡

Telegram + MCP

Built-in Telegram bot plugin. MCP server declarations per project.

🧠

Multi-engine

Anthropic, OpenAI, Gemini, Ollama, or mock. Configured per project or globally.

📋

Activity log

Every invocation logged to JSONL. Stream with apx messages tail.

Quick start

apx (local)
$ apx agent list
Projects:
· Project A .apc/
· Project B .apc/

$ apx run superagent --runtime claude-code "Review open PRs"
