## ¿Qué es APX?

APX es la implementación de referencia del [protocolo APC](https://github.com/agentprojectcontext/agentprojectcontext):
un **daemon local + CLI `apx`** que gestiona proyectos, agentes, sesiones, memoria y registros de
mensajes. El sistema de archivos es la fuente de verdad — el contexto del proyecto vive en tu repo, el
estado del runtime vive en `~/.apx/`.
